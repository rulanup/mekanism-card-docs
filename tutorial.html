安装方法

1. 准备前置模组

  1. 安装 Minecraft 1.21.1 与 NeoForge 21.1.220+
  2. 安装 Mekanism 10.7.14+
  3. 安装 GuideME 21.1.15+
  4. （可选）安装 AE2 19.2+ 以使用网络联动

2. 放入 mods 文件夹

  1. 下载 Mekanism Card 模组 jar 文件
  2. 将 jar 文件放入 .minecraft/mods 文件夹
  3. 启动游戏，进入存档
  4. 在创造模式物品栏或合成表中查找“Mekanism 升级卡”分类

小提示

首次使用建议先合成一本 指南书（书 + 批量升级配置器），右键即可在游戏内查看详细说明。

快速上手

绑定网络存储

许多工具支持从 AE2 / QIO 抽取物品，绑定方式如下：

  • AE2：将工具放入无线访问点的链接槽位
  • QIO：潜行 + 右键已选择频道的 QIO 方块
  • 消耗顺序：背包 > AE2 > QIO

通用操作

  • 右键空气：切换工具模式
  • 潜行 + 右键空气：切换半径 / 选区模式
  • 中键机器：自动安装所有支持的升级
  • 右键机器：执行当前模式的主要操作

工具详解

下面分别介绍四件核心工具的使用方法、注意事项与进阶技巧。

终极等级安装器

一键将任意 Mekanism 机器升级到终极等级。内置 200,000 FE 能量，每次升级消耗 1,000 FE。如果安装了通用机械扩展，还支持继续升级到绝对、至尊、宇宙、无限等级。

使用方法

  • 右键机器：将单台机器升级到终极等级
  • Ctrl + 右键：切换区域升级模式，右键后升级所有相连机器
  • 潜行 + 右键 QIO 方块：绑定 QIO 频道
  • 将工具放入无线访问点链接槽：绑定 AE2 网络

注意事项

  • 升级物品优先从背包消耗，不足时自动从 AE2 / QIO 抽取
  • 区域模式下，所有相连机器会逐级消耗对应安装器
  • 确保工具能量充足，否则无法执行升级

批量升级配置器

批量为 Mekanism 机器安装或移除升级模块。工具会根据背包、AE2、QIO 中的可用升级自动检测并执行操作。

模式切换

  • 右键空气：切换安装 / 移除模式
  • 潜行 + 右键空气：切换半径模式 / 选区模式

执行操作

  • 半径模式：潜行 + 右键机器，对所有相邻连接的同类型机器执行操作
  • 选区模式：潜行 + 右键设置两个角点，再右键机器对选区内所有机器执行操作
  • 中键机器：自动为该机器安装所有支持的升级模块（每种升到满级）

视觉反馈

  • 绿色轮廓 = 安装模式
  • 红色轮廓 = 移除模式
  • 灰色轮廓 = 无可用升级

存储卡

用于在 Mekanism 机器之间复制和粘贴配置与升级。首次右键机器会复制配置，再次右键同类型机器会粘贴配置。

使用方法

  • 右键机器（无数据时）：复制该机器的配置与升级
  • 右键同类型机器（有数据时）：粘贴配置，必要时消耗升级模块
  • 潜行 + 右键空气：清除已保存的配置

注意事项

  • 目标机器类型必须与源机器一致，否则会提示“机器类型不匹配”
  • 升级模块可从绑定的 AE2 / QIO 网络中自动抽取
  • 创造模式下粘贴配置不消耗任何升级材料

超级融合卡

将等级安装、升级模块安装和存储卡复制 / 粘贴功能融合于一体的终极工具。适合已经熟悉前三件工具的玩家，一个物品即可应对所有批量场景。

快捷键

  • Ctrl + 右键空气：在等级安装 / 安装升级 / 内存复制粘贴之间切换
  • Ctrl + Shift + 右键空气：切换范围 / 单个模式
  • Shift + 右键空气：切换模块选区模式
  • Alt + 右键空气：清除已保存的内存数据
  • 右键机器：执行当前融合模式（会阻止机器界面打开）
  • 中键机器：自动安装所有支持的升级模块

融合模式

  • 等级安装：将机器向终极等级升级
  • 安装升级：安装或移除升级模块，右键空气切换安装 / 移除
  • 内存复制 / 粘贴：无数据时复制，有数据时粘贴

网络联动

与终极等级安装器和批量升级配置器相同，超级融合卡可以从绑定的 AE2 网络或 QIO 频道抽取等级安装器和升级模块。

使用技巧

优先绑定网络

后期工厂通常将升级物品存放在 AE2 或 QIO 中。提前绑定网络后，工具会自动从网络抽取，无需随身携带成组的安装器。

善用选区模式

当机器呈规则阵列摆放时，使用选区模式可以精确控制影响范围，避免误操作相邻的其他设备。

中键快捷升级

不确定某台机器支持哪些升级？直接中键点击，工具会自动计算并安装所有支持的升级模块到满级。

能量管理

终极等级安装器和超级融合卡都需要 FE 能量。建议在基地常备能量立方或无线充电设备，操作前检查电量。
