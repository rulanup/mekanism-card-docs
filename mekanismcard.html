Mekanism Card
通用机械卡牌

通用机械模组的批量操作工具。在建造大型工厂时，为你节省大量繁琐的点击操作，让机器升级与配置管理变得轻而易举。

批量升级 终极等级安装 配置复制 / 粘贴 AE2 / QIO 联动
查看使用教程 GitHub 仓库

四大核心工具

从一键升级到批量配置，Mekanism Card 为每种重复操作提供了趁手的解决方案。

终极等级安装器

右键任意 Mekanism 机器，即可一键将其升级到终极等级，支持 AE2 / QIO 网络自动抽取升级物品。

  • 单次 / 区域两种升级模式
  • 内置 200,000 FE 能量缓存
  • 消耗优先级：背包 > AE2 > QIO

批量升级配置器

一次性为区域内所有相连机器安装或移除速度、能源、降噪等升级模块，告别逐台操作的烦恼。

  • 半径模式与选区模式
  • 中键快捷安装所有支持升级
  • 彩色轮廓实时显示操作状态

存储卡

像复制粘贴文本一样复制 Mekanism 机器的配置与升级，快速将一台机器的设置同步到多台同类型机器。

  • 右键复制 / 粘贴配置
  • 自动批量应用到相连机器
  • 创造模式粘贴不消耗材料

超级融合卡

集等级安装、模块安装、配置复制粘贴于一体的全能工具，一个物品搞定所有批量操作。

  • 三种融合模式自由切换
  • 中键自动安装满级升级
  • 从绑定网络抽取所需物品

为什么需要 Mekanism Card？

在大型自动化工厂中，升级数百台机器、统一配置参数是重复又耗时的工作。Mekanism Card 将这些操作浓缩为几次点击。

⚡ 节省时间

批量安装升级、批量粘贴配置、区域终极升级，将原本需要数十分钟的工作压缩到几秒。

🔗 网络联动

与 AE2 和 Mekanism QIO 深度集成，升级物品可直接从网络中抽取，无需随身携带大量安装器。

🎯 精准操作

支持半径模式和自定义长方体选区，只对你想操作的机器生效，避免误操作。

📘 游戏内指南

内置基于 GuideME 的指南书，随时右键查看各工具的详细使用说明。

运行要求

在开始使用前，请确认你的游戏环境满足以下前置条件。

必需前置

  • Minecraft 1.21.1
  • NeoForge 21.1.220+
  • Mekanism 10.7.14+
  • GuideME 21.1.15+

可选联动

  • AE2 19.2+ — 从 AE2 网络抽取升级物品
  • Mekanism QIO — 从 QIO 频道抽取升级物品
  • 通用机械扩展 — 终极等级安装器支持继续升级到无限等级

安装提示

将模组 jar 文件放入 .minecraft/mods 文件夹，并确保已安装上述前置模组，启动游戏即可。详细步骤请参考 安装指南。
